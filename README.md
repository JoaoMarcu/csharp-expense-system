# csharp-expense-system
# 💸 Controle de Gastos (Console App em C#)

Um sistema simples de controle de gastos desenvolvido em C#, utilizando um CRUD completo no terminal.

## 📌 Funcionalidades

* ➕ Adicionar gastos
* 📋 Listar gastos
* ✏️ Atualizar gastos
* ❌ Remover gastos
* 💰 Calcular total gasto
* 📅 Registro automático da data

## 🧠 Conceitos aplicados

* Estruturas de repetição (`while`)
* Estruturas condicionais (`switch`)
* Listas (`List<T>`)
* Programação orientada a objetos
* Métodos
* Validação de entrada (`TryParse`)
* Manipulação de datas (`DateTime`)

## ⚙️ Tecnologias

- **C#**  
- **.NET** 

## ▶️ Como Executar

```bash
# Clonar o repositório
git clone https://github.com/JoaoMarcu/csharp-expense-system.git

# Entrar na pasta do projeto
cd csharp-expense-system

# Executar o projeto
dotnet run
```

## 📷 Exemplo de uso

```plaintext
1 - Adicionar gasto
2 - Listar gastos
3 - Atualizar gasto
4 - Remover gasto
5 - Total gasto
0 - Sair
```
